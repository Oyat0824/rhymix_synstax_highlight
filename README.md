# Synstax Highlight

코드 문법 강조 에디터 컴포넌트입니다. Highlight.js를 사용하여 다양한 프로그래밍 언어의 코드를 하이라이트 처리합니다.

## 경로

```
/modules/editor/components/synstax_highlight
```

## 설치 방법

1. 이 컴포넌트 폴더를 라이믹스 설치 경로의 `/modules/editor/components/` 디렉토리에 복사합니다.
2. 관리자 페이지에서 에디터 설정으로 이동합니다.
3. 컴포넌트 설정에서 테마와 사용할 언어를 선택합니다.

## 사용법

1. 에디터에서 코드 블록을 삽입할 위치에 커서를 둡니다.
2. 에디터 도구 모음에서 코드 블록 버튼을 클릭합니다.
3. 팝업 창에서 언어를 선택하고 코드를 입력합니다.
4. 미리보기로 확인 후 삽입합니다.

## 지원 언어

JavaScript, Python, Java, PHP, C/C++, XML/HTML, CSS, SQL, JSON, Bash 등 20여 개 언어를 지원합니다.
